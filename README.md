# Atlantic Ukrainian Capital Club Website

Static multi-page site built for GitHub Pages (no build step).

## Pages
- `/` — Home
- `/about/` — About
- `/events/` — Events
- `/membership/` — Membership
- `/contact/` — Contact

## Tech
- HTML
- CSS (`/assets/css/styles.css`)
- Vanilla JS (`/assets/js/main.js`)

## Notes
- English-only primary experience.
- Legacy bilingual drafts remain in `/en` and `/ua`, but the new navigation only links to the new top-level pages.
- `CNAME` is preserved for custom domain deployment.
